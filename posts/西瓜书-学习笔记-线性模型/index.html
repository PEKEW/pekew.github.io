西瓜书 学习笔记 线性模型

2022-10-19
5 min read
AI
ZhiHuaZhou-MachineLearning-Reading

总的来说，线性模型分为线性回归模型和线性分类模型，线性模型指的是我们试图学习到一个属性的线性组合来进行预测的函数以完成不同的任务，个人认为，线性模型是最简单的机器学习模型，但其思想能够代表一众复杂的机器学习模型。

我们首先给出线性模型的一般(或者叫广义)线性模型的定义:
$$y = g(\omega^Tx+b)$$ 其中,$g$连续，单调可微分且充分光滑。$g$起到的作用类似于激活函数，给模型一个非线性的映射。值得注意的是，很多人会把线性模型的线性理解为学习到的目标函数，或者模型在解空间的映射必须是一条直线，但我们自己阅读线性模型的文字部分定义就不难发现：线性模型中的线性指的是“特征”的线性组合，而和解空间形状无关。

线性模型形式简单，建模方便，可解释性好，一定程度上是非线性模型的基础。

线性模型用于回归任务

用于回归任务的线性模型叫做线性回归。用公式表示为： $$f(x_i) = \omega x_i + b_i$$ 其中,$\omega,b$是线性模型的参数，对线性模型的学习或训练过程就是对这两个参数估计的过程。

最小二乘法

对参数的估计过程实际上就是找到最合适的参数，使得线性模型的输出值和真实值之间的差别尽可能小，衡量这种差别最常用的方法是均方误差。我们完成均方误差的最小化实际上就完成了对线性模型的参数估计。

最小二乘法指的是基于均方误差最小化来求解模型参数的方法。

由于均方误差构造了一个关于$\omega, b$的凸函数，因此可以求得最优闭式解。

关于最小二乘法的推导过程暂且不表（打字太多）

线性模型的分类问题

考虑二分类问题，我们常设置分类阈值，当预测结果大于阈值时为某一类，而小于某阈值时为另一类。这是十分直观的做法，因此，基于这个思路，我们需要对分类模型的输出加以调整，使其满足分类要求。理想情况下，我们只要输出0/1即可完成二分类，于是最理想的激活函数为线性阶跃函数: $$y = \begin{cases} 0, z < 0 \ 0.5,z=0 \ 1,z>0 \end{cases}$$

显然，这个函数不连续，那只能退而求其次。最简单的，我们可以试图把模型的输出压缩到[0,1]区间，这样输出结果可以代表样本属于某个类的概率。基于这样的思路，我们在分类问题中常用S函数代替线性阶跃函数。 $$y = \frac{1}{1+e^{-z}}, z = \omega^Tx+b$$ 对上式进行恒等变换，左侧变为$ln\frac{y}{1-y}$。上文提到，我们把模型的输出结果看作是属于某个类的概率，或者说“可能性”。那$ln\frac{y}{1-y}$可以看做两个类别的相对可能性，称为"几率",取对数则为对数几率。

对数几率回归

对数几率回归实际上是一种线性回归模型变化而来的分类模型。有几大优点：

  • 对可能性直接建模，无需假设分布，避免了假设不准确带来的问题。
  • 不仅是预测类别，而是预测概率。
  • 对数几率是任意阶可到的凸函数，数学性质好。有很多数值优化方法可以直接求解最优解。

对数几率回归的参数估计

把对数几率回归的模型中的输出$y$作为后验概率估计，则其可以变形为: $$ln\frac{p(y=1|x)}{p(y=0|x)} = \omega^Tx+b$$ 对应地: $$p(y=1|x) = \frac{e^{\omega^Tx+b}}{1+e^{\omega^Tx+b}}$$ $$p(y=0|x) = \frac{1}{1+e^{\omega^Tx+b}}$$ 可以通过最大似然法对参数进行估计。 在此略。

线性判别分析

线性判别分析的基本思想很简单，把数据映射到一条直线上，使得不同类别之间的差距（类间距）尽可能大，而相同类之间的差距（类内距）尽可能小。

先考虑怎么衡量类内距和类间距，类间距很简单：两个类别投影的中心点之间的差距: $$|\omega^T\mu_0 - \omega^T\mu_1|$$ 其中,$\mu_0,\mu_1$是两个类别投影过后的中心点。$\omega$是映射变化。 至于类内距，我们考虑协方差矩阵，协方差可以衡量两个的总体误差，所以使用协方差矩阵来衡量类内距: $$\omega^T\Sigma_0\omega + \omega^T\Sigma_1\omega$$ 线性判别分析的目的是类内距小，类间距大，即如下最小化： $$\min J = \frac{\omega^T(\Sigma_0+\Sigma_1)\omega}{|\omega^T\mu_0 - \omega^T\mu_1|}$$

这个优化目标可以改写成一个瑞利商的形式。使用拉格朗日乘子法求解最优映射。（过程略） 需要注意的是，线性判别分析只关注映射的方向，并不考虑大小。

当线性判别分析用于多分类时，原本的$\omega$变为投影矩阵，其闭式解是$N-1$维的，因此，线性判别分析也可以用来降维，是一种经典的监督降维技术。

多分类

多分类任务其实可以使用二分类模型来解决多分类问题。

可以把多分类的多个类别拆解为多个二分类的问题。经典的策略有三种：

  • 一对一：两两组合,分别作为正反例，预测时先产生所有的分类器的预测结果，然后进行投票。
  • 一对其余：一类作为正例，其余作为反例。相比于一对一方法，一对其余的拆分方法需要训练的模型更少，但每个模型训练开销更多。预测性能和一对一不相上下。
  • 多对多：若干为正，若干为反，而且正反类的设计需要进行特殊的构造。不能随意。经典的多对多技术有ECOC（纠错输出码），纠错输出码能够对分类器的错误有一定的容忍和修正能力。一般来说，对于同一个分类任务，编码越长，纠错能力越长，需要训练的分类器也就越多。

类别不平衡问题

在一般的二分类问题中，我们常设置0.5为判别阈值，这隐藏了一个假设，即就是我们认为训练样本中的正例和反例分布是一致的，实际上经常会遇到分类类别不平衡的问题，有如下解决方案：

  • 再缩放：实际上也是代价敏感学习。设置阈值为训练样本中的正例和反例的比例，实际上再缩放的本质是我们假设观测几率代表真是几率。
  • 欠采样：去除反例，使得正反样本数量接近，但是欠采样可能会丢失一些重要要本，使得丢失重要信息。典型的欠采样方法是利用集成学习机制，划分若干个集合供不同的模型训练，对于这些模型来说，都是欠采样处理的。
  • 过采样：和欠采样相反，过采样是增加一些正例，使得正反例样本数量接近。但开销也大，而且不能简单地进行重复采样，否则会严重过拟合，因此，经典的过采样方法是进行插值以生成额外的正例。
  • 阈值移动：在预测过程中使用再缩放，因为简单的再缩放的假设“无偏采样”很难做到。
